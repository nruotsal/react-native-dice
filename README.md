React Native Dice
===

## Prerequisites

* [Node.js](https://nodejs.org/en/) version 12 or later.
* [Expo](https://expo.io/)

```sh
npm install expo-cli -g
```

## Development

Install dependencies

```sh
yarn install
```

Start Expo

```sh
expo start
```

## Testing

Run linter

```sh
yarn lint
```

Run tests

```sh
yarn test
```
